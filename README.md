# Food oredering app
Spring boot Angular Food oredering app

## Features
- REST API
- JWT authentication
- Cookie based visitors' shopping cart
- Persistent customers' shopping cart
- Cart & order management
- Order management
- Pagination
- Adding new Food items
- Swagger UI on link http://localhost:8080/swagger-ui/
